Про EasyWords

Твій простий та зручний помічник для вивчення англійських слів

EasyWords — це сучасний та зручний веб-додаток, розроблений спеціально для ефективного вивчення англійської лексики.

Наша головна мета відмовитись від паперових нотатків і зберігати всі нові слова в одному місці.

Основні можливості:

  • Створення власних карток зі словами
  • Зручний перегляд та редагування списку
  • Доступ до словника з будь-якого пристрою
  • Простий та інтуїтивно зрозумілий інтерфейс

Розпочати навчання дуже просто:

  • Авторизація:
    Створіть акаунт або увійдіть, щоб ваші слова зберігалися в хмарі.
  • Пошук та додавання слів:
    Шукайте вже збережені слова через рядок пошуку, або натисніть Додати, щоб записати нове слово.
  • Регулярність:
    Заходьте у додаток щодня, переглядайте та тренуйте пам'ять.
  • Результат:
    Вже за місяць Ваш словниковий запас значно зросте!

Цей проект розроблено з використанням сучасного стеку веб-технологій (Frontend), що забезпечує швидку роботу та адаптивність.

Основний стек:
  • HTML5 — семантична структура сторінок.
  • Bootstrap 5.3 — потужний CSS-фреймворк для стилізації.
Дизайн та UI:
  • Mobile First — повна адаптація під смартфони.
  • Bootstrap Icons — векторна графіка та іконки.
Проект створено в середовищі Visual Studio Code в рамках лабораторної роботи з курсу "Web-дизайн"
